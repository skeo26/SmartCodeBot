Question: Для чего используется оператор ORDER BY в SQL?
Options:
- Для фильтрации строк по условию
- Для сортировки результатов запроса
- Для объединения таблиц

Explanation: ORDER BY используется для сортировки строк по одному или нескольким столбцам.

- Сортировка не произойдёт
- Будет использована сортировка по убыванию
- Будет использована сортировка по возрастанию
