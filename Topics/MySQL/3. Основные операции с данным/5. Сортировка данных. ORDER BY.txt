Оператор <code>ORDER BY</code> используется для сортировки строк, возвращаемых оператором <code>SELECT</code>. Сортировка может быть по одному или нескольким столбцам, в порядке возрастания или убывания.

<b>Синтаксис:</b>
<code>
SELECT столбцы FROM таблица ORDER BY столбец [ASC|DESC];
</code>

<b>Пояснение:</b>
- <code>ASC</code> — сортировка по возрастанию (по умолчанию)
- <code>DESC</code> — сортировка по убыванию

<b>Пример:</b>
<code>
SELECT * FROM users ORDER BY age ASC;
</code>
Выведет всех пользователей, отсортированных по возрасту от младших к старшим.

<b>Сортировка по нескольким столбцам:</b>
<code>
SELECT * FROM users ORDER BY city ASC, age DESC;
</code>
Сначала сортирует по городу в алфавитном порядке, затем по возрасту — от старших к младшим внутри каждого города.

<b>Полезно знать:</b>
- Сортировка влияет только на порядок вывода данных, сами данные в таблице не меняются.
- Можно использовать <code>ORDER BY</code> с любыми столбцами, не обязательно выводимыми через <code>SELECT</code>.
