Операция <code>SELECT</code> используется для извлечения данных из базы данных. Это одна из самых часто используемых команд в SQL.

<b>1. Основной синтаксис:</b>
<code>
SELECT столбцы FROM таблица;
</code>

<b>Пример:</b>
<code>
SELECT * FROM users;
</code>
Этот запрос выбирает все данные из таблицы <code>users</code>.

<b>2. Выбор отдельных столбцов:</b>
Если нужно выбрать конкретные столбцы, указывайте их через запятую:
<code>
SELECT name, age FROM users;
</code>
Этот запрос выбирает только столбцы <code>name</code> и <code>age</code> из таблицы <code>users</code>.

<b>3. Фильтрация данных с помощью WHERE:</b>
Чтобы извлечь данные, соответствующие определенным условиям, используйте оператор <code>WHERE</code>:
<code>
SELECT * FROM users WHERE age > 18;
</code>
Этот запрос выбирает все строки из таблицы <code>users</code>, где возраст больше 18 лет.

<b>4. Сортировка результатов:</b>
Чтобы отсортировать результаты, используйте оператор <code>ORDER BY</code>:
<code>
SELECT * FROM users ORDER BY age DESC;
</code>
Этот запрос выбирает все строки из таблицы <code>users</code> и сортирует их по столбцу <code>age</code> в порядке убывания (DESC).

<b>5. Ограничение количества строк:</b>
Чтобы ограничить количество возвращаемых строк, используйте оператор <code>LIMIT</code>:
<code>
SELECT * FROM users LIMIT 5;
</code>
Этот запрос выберет только первые 5 строк из таблицы <code>users</code>.

<b>6. Использование алиасов для таблиц и столбцов:</b>
Можно использовать алиасы для удобства работы с длинными именами таблиц и столбцов:
<code>
SELECT u.name, u.age FROM users AS u;
</code>
Здесь таблице <code>users</code> присваивается алиас <code>u</code>, и в запросе можно использовать его вместо полного имени.

<b>7. Пример с несколькими условиями:</b>
Для сложных условий используйте логические операторы <code>AND</code> и <code>OR</code>:
<code>
SELECT * FROM users WHERE age > 18 AND city = 'Moscow';
</code>
Этот запрос выбирает пользователей старше 18 лет, которые живут в Москве.
