Операции <code>UPDATE</code> и <code>DELETE</code> используются для изменения и удаления данных в базе данных соответственно.

<b>1. Операция UPDATE:</b>

Синтаксис:
<code>
UPDATE таблица SET столбец1 = значение1, столбец2 = значение2 WHERE условие;
</code>

Пример:
<code>
UPDATE users SET age = 25 WHERE name = 'Alice';
</code>
Этот запрос обновляет возраст пользователя с именем "Alice" на 25 лет.

<b>Важно:</b> Если не использовать условие <code>WHERE</code>, все строки таблицы будут обновлены.

<b>2. Операция DELETE:</b>

Синтаксис:
<code>
DELETE FROM таблица WHERE условие;
</code>

Пример:
<code>
DELETE FROM users WHERE name = 'Bob';
</code>
Этот запрос удаляет пользователя с именем "Bob" из таблицы <code>users</code>.

<b>3. Важные замечания:</b>
- Всегда используйте условие <code>WHERE</code> при удалении данных, чтобы избежать удаления всех строк из таблицы.
- Для массового обновления данных можно использовать различные логические операторы в условии <code>WHERE</code>.

<b>Пример:</b> 
Обновление статуса пользователей:
<code>
UPDATE users SET status = 'active' WHERE last_login &gt; '2022-01-01';
</code>

<b>Пример:</b>
Удаление пользователей, не активных более года:
<code>
DELETE FROM users WHERE last_login &lt; '2023-01-01';
</code>
