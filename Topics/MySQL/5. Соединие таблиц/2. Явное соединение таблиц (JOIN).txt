<b>Явное соединение</b> — это современный и предпочтительный способ объединения данных из нескольких таблиц с использованием ключевого слова <code>JOIN</code>.

<b>Виды JOIN:</b>
- <code>INNER JOIN</code> — соединяет только те строки, у которых есть совпадения в обеих таблицах.
- <code>LEFT JOIN</code> — возвращает все строки из левой таблицы и совпадающие строки из правой.
- <code>RIGHT JOIN</code> — возвращает все строки из правой таблицы и совпадающие строки из левой.
- <code>FULL JOIN</code> (не поддерживается напрямую в MySQL) — возвращает все строки из обеих таблиц, даже если нет совпадений.

<b>Синтаксис INNER JOIN:</b>
<code>
SELECT таблица1.поле, таблица2.поле
FROM таблица1
JOIN таблица2 ON таблица1.ключ = таблица2.ключ;
</code>

<b>Пример:</b>
<code>
SELECT employees.name, departments.name
FROM employees
JOIN departments ON employees.department_id = departments.id;
</code>

<b>LEFT JOIN:</b>
<code>
SELECT employees.name, departments.name
FROM employees
LEFT JOIN departments ON employees.department_id = departments.id;
</code>

Этот запрос выведет всех сотрудников, даже если у них не указано подразделение.

<b>Преимущества явного JOIN:</b>
- Чёткое разделение логики соединения и фильтрации.
- Удобство для чтения и поддержки кода.
- Совместимость с большинством современных СУБД.

<b>Важно:</b>
- Всегда указывайте условие соединения через <code>ON</code>, иначе результат может быть неожиданным.
