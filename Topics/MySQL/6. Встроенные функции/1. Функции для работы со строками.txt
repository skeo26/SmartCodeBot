MySQL предоставляет множество встроенных функций для работы с строками. Эти функции позволяют изменять, анализировать и форматировать строки данных.

<b>Основные функции для работы со строками:</b>

1. <code>CONCAT()</code> — объединяет несколько строк в одну.
   Пример:
   <code>
   SELECT CONCAT('Hello', ' ', 'World!');
   -- Результат: 'Hello World!'
   </code>

2. <code>LENGTH()</code> — возвращает длину строки в байтах.
   Пример:
   <code>
   SELECT LENGTH('Hello');
   -- Результат: 5
   </code>

3. <code>CHAR_LENGTH() или LENGTH()</code> — возвращает количество символов в строке (игнорирует байтовое представление).
   Пример:
   <code>
   SELECT CHAR_LENGTH('Hello');
   -- Результат: 5
   </code>

4. <code>SUBSTRING()</code> — извлекает подстроку из строки.
   Пример:
   <code>
   SELECT SUBSTRING('Hello', 2, 3);
   -- Результат: 'ell' (начиная с позиции 2, длина 3)
   </code>

5. <code>TRIM()</code> — удаляет пробелы с обеих сторон строки.
   Пример:
   <code>
   SELECT TRIM('  Hello  ');
   -- Результат: 'Hello'
   </code>

6. <code>UPPER()</code> — преобразует строку в верхний регистр.
   Пример:
   <code>
   SELECT UPPER('hello');
   -- Результат: 'HELLO'
   </code>

7. <code>LOWER()</code> — преобразует строку в нижний регистр.
   Пример:
   <code>
   SELECT LOWER('HELLO');
   -- Результат: 'hello'
   </code>

8. <code>REPLACE()</code> — заменяет все вхождения подстроки в строке.
   Пример:
   <code>
   SELECT REPLACE('Hello World', 'World', 'MySQL');
   -- Результат: 'Hello MySQL'
   </code>

9. <code>CONCAT_WS()</code> — объединяет строки, разделяя их указанным разделителем.
   Пример:
   <code>
   SELECT CONCAT_WS('-', '2025', '04', '11');
   -- Результат: '2025-04-11'
   </code>

10. <code>INSTR()</code> — возвращает позицию первого вхождения подстроки в строке.
    Пример:
    <code>
    SELECT INSTR('Hello World', 'World');
    -- Результат: 7 (позиция первого вхождения 'World')
    </code>

11. <code>LEFT()</code> — возвращает указанное количество символов с левой стороны строки.
    Пример:
    <code>
    SELECT LEFT('Hello', 3);
    -- Результат: 'Hel'
    </code>

12. <code>RIGHT()</code> — возвращает указанное количество символов с правой стороны строки.
    Пример:
    <code>
    SELECT RIGHT('Hello', 3);
    -- Результат: 'llo'
    </code>

<b>Дополнительные функции:</b>

- <code>REVERSE()</code> — переворачивает строку.
- <code>FIELD()</code> — возвращает позицию первого вхождения значения в списке значений.
- <code>SOUNDEX()</code> — возвращает звуковой код для строки (используется для поиска похожих по звучанию строк).
