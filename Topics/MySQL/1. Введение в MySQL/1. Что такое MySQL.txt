<b>Что такое MySQL?</b>
MySQL — это популярная система управления базами данных с открытым исходным кодом. Она используется для хранения, управления и извлечения данных. Наиболее часто применяется в веб-разработке, где MySQL работает в связке с языками программирования, например, C# или Python.

<b>Основные особенности MySQL:</b>
- Высокая производительность и надежность.
- Поддержка языка SQL (Structured Query Language) — стандарта для работы с базами данных.
- Возможность работы с большими объемами данных.
- Расширяемость и поддержка хранимых процедур, триггеров и пользовательских функций.
- Бесплатность и открытый исходный код (GPL лицензия).
- Кроссплатформенность — работает на Windows, Linux, macOS и других ОС.

<b>Где применяется:</b>
- Веб-сайты и CMS (например, WordPress, Joomla).
- Интернет-магазины.
- Мобильные и настольные приложения.
- Корпоративные системы учета и аналитики.

<b>Пример простой SQL-запроса:</b>

<code>
SELECT * FROM users WHERE age &gt; 18;
</code>

Этот запрос выберет всех пользователей из таблицы <code>users</code>, у которых возраст больше 18 лет.
