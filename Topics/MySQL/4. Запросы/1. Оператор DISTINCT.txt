Оператор <code>DISTINCT</code> используется в SQL-запросах для удаления дублирующихся строк из результата. Он позволяет вывести только уникальные значения столбца или комбинации столбцов.

<b>Синтаксис:</b>
<code>
SELECT DISTINCT столбец1, столбец2, ...
FROM таблица;
</code>

<b>Пояснение:</b>
- <code>DISTINCT</code> применяется ко всем перечисленным столбцам.
- Если хотя бы одно значение в строке отличается, строка считается уникальной.

<b>Пример 1: Уникальные значения одного столбца</b>
<code>
SELECT DISTINCT city FROM users;
</code>
Выведет список всех уникальных городов, где проживают пользователи.

<b>Пример 2: Уникальные комбинации</b>
<code>
SELECT DISTINCT city, age FROM users;
</code>
Выведет уникальные пары "город + возраст".

<b>Полезно знать:</b>
- <code>DISTINCT</code> используется только с оператором <code>SELECT</code>.
- Может быть затратным по производительности на больших объемах данных, так как требует сортировки или хеширования.
